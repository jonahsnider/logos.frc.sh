# logos.frc.sh - FRC team logos

**[logos.frc.sh](https://logos.frc.sh)**

A web app for browsing FRC team logos.

## Development

logos.frc.sh was built with [Tanstack Start](https://tanstack.com/start/latest) and uses [shadcn/ui](https://ui.shadcn.com/) and [Tailwind CSS](https://tailwindcss.com/) for styling.
